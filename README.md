Public portfolio website for Hamza Javaid, focused on video editing, motion design, generative AI video workflows, and performance-focused creative production.

## Live Site

This repository is prepared for GitHub Pages. The website entry file is `index.html`.

## Contents

- `index.html` - main portfolio page
- `splash-background-effect.js` - custom cursor/splash interaction layer
- `hamza-logo.svg` - portfolio logo asset
- `assets/` - portrait, company logos, value icons, and process icons
